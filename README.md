# synent-task5-salesanalysis-bhupendra
Sales analysis project using the Superstore Dataset. Performed data cleaning, exploratory data analysis (EDA), visualization, and business insight generation. Analyzed revenue trends, top-selling products, category performance, and regional profitability using Python, Pandas, Matplotlib, and Seaborn.

## Dataset Description

### Superstore Dataset

The Superstore Dataset is a retail business dataset commonly used for data analytics, business intelligence, and data science projects. It contains information about customer orders, sales performance, product categories, profits, discounts, shipping details, and regional operations.

This dataset is designed to help analyze business performance and identify opportunities for improving sales and profitability.

### Dataset Source

Kaggle – Sample Superstore Dataset

Source Link:
https://www.kaggle.com/datasets/vivek468/superstore-dataset-final

### Dataset Overview

The dataset contains transactional sales records from a retail superstore.

Key information includes:

* Customer details
* Product information
* Sales transactions
* Profit and loss records
* Regional performance
* Shipping details
* Order dates

### Important Features

| Feature       | Description                                         |
| ------------- | --------------------------------------------------- |
| Order ID      | Unique identifier for each order                    |
| Order Date    | Date when the order was placed                      |
| Ship Date     | Date when the order was shipped                     |
| Customer Name | Customer associated with the order                  |
| Segment       | Customer segment (Consumer, Corporate, Home Office) |
| Country       | Country of operation                                |
| City          | Customer city                                       |
| State         | Customer state                                      |
| Region        | Sales region                                        |
| Product ID    | Unique product identifier                           |
| Category      | Product category                                    |
| Sub-Category  | Product sub-category                                |
| Product Name  | Product description                                 |
| Sales         | Revenue generated from the order                    |
| Quantity      | Number of units sold                                |
| Discount      | Discount applied                                    |
| Profit        | Profit earned from the transaction                  |
